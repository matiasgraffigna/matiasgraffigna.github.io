# Traducciones
Realizo traducciones técnicas al español de inglés y alemán. Para traducir artículos, libros o textos, puede solicitar un presupuesto. Envío muestra de trabajo de tres páginas y honorarios correspondientes.

# Correcciones
Realizo correcciones de estilo, ortografía y gramática. Miembro de PLECA. Para consultas, envíe su texto o parte y realizo una muestra de trabajo de tres páginas y un presupuesto. Puede ver las tarifas de referencia aquí (estas varían según el tipo de trabajo).
